## 导出文件结构

| Option | Description | Selected |
|--------|-------------|----------|
| 单个汇总表 | 简单，但不利于核对明细和支出 | |

## 月度明细表头

| Option | Description | Selected |
|--------|-------------|----------|
| 财务软件式抽象报表 | 字段规范，但不贴近日常房租水电核账 | |
| 参考“每月房租、水电收入及各项支出”表头 | 更贴近用户现有算账方式，一行一个房间 | ✓ |

**User's choice:** 用户提供图片表头作为参考，按推荐扩展。  
**Notes:** 表头包括序号、水（上月）、水（本月）、实用（方）、水费、电（上月）、电（本月）、实用（度）、电费、房租（元）、维修费、其他费用、退租支出、房租水电合计等。

## 统计口径

| Option | Description | Selected |
|--------|-------------|----------|
| 只按应收月份 | 适合看应收和欠收，但看不到真实到账月份 | |
| 只按实收月份 | 适合看现金流，但看不到本月应收 | |
| 同时保留应收月份和实收日期 | 既能看本月应收，也能查实际到账 | ✓ |

**User's choice:** 按推荐。  
**Notes:** 月度汇总展示本月应收、本月实收、截至导出仍未收；房东支出按发生日期归月。
