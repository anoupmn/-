# 07-04 凭证体验同步口径

## Goal

把收据详情打磨成可打印、可分享、可管理的正式收款凭证，并完成 Phase 07 回归。

## Final Scope

- 收据标题固定为 `收款收据（非发票）`。
- 凭证内容：收据编号、房源/房间、租客、收款日期、收款项目明细、合计金额、备注、生成时间。
- 不展示收款人字段。
- 明细按收据快照 `items` 渲染，包含项目名称、应收日期、实收日期、金额。
- PDF 打印版使用稳定表格排版，日期和金额不得重叠。
- 收据详情动作：导出 PDF 打印版、复制收据摘要、分享凭证、删除收据。
- 收据记录页按月份自动整理，卡片可查看、回房间、删除。
- 部署清单覆盖所有收据相关云函数。

## Manual UAT

1. 业务维护 -> 收据记录 -> 查看收据。
2. 收据记录 -> 按租约开收据 -> 选择租约和月份 -> 收据预览。
3. 房间详情 -> 开具本租约本月收据 -> 收据预览。
4. 收据预览 -> 导出 PDF 打印版。
5. 收据记录/收据详情 -> 删除收据 -> 回到房间重新开具。
6. 收据预览 -> 复制收据摘要 / 分享凭证。
